# Lantliv — projektinstruktion

Det här är **Lantliv**, familjens co-op farming→bybygge-spel (Little Dreamyland-sprites).

**För full kontext och att återuppta arbetet: läs [SESSION.md](SESSION.md).**
Den innehåller status, filkarta, nästa steg (Fas B) och alla designbeslut.

- Live: https://8bitcat.github.io/lantliv · Repo: `main` → GitHub Pages (deploy = push).
- Buildless HTML5 Canvas + ES-moduler, ingen bygg. Lokal test: `python -m http.server 8137`.
- Verifiera alltid ändringar med en headless-render/screenshot innan commit + push.
